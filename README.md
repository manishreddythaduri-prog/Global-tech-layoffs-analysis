# Global-tech-layoffs-analysis
This project analyzes global technology layoffs by different companies and countries using python.
The goal is to clean the dataset, explore layoff trends, visualize key patterns, and generate business insights.
This project is just created as a portfolio project for showcase my skills.

# Main Objectives are :
- Clean and prepare layoff data
- Analyze data by year and month
- identify companies with highest layoffs
- identify countries with more layoffs
- identify industries most affected by layoffs
- Create clear data visualization
- Which Year has highest layoffs
- summarize business insights

# The dataset contains 
- information about global technology layoffs, datset columns include :
'Company', 'Location_HQ', 'Region', 'USState', 'Country', 'Continent', 'Laid_Off', 'Date_layoffs', 'Percentage', 'Company_Size_before_Layoffs', 'Company_Size_after_layoffs', 'Industry', 'Stage', 'Money_Raised_in__mil', 'Year'.

Dataset resource : Kaggle/Ulrike Herold/Tech Layoffs 202-25

# Tools used :
Python, Pandas, Numpy, Matplotlib, Seaborn, Visual studio code

# key insights
- The datset shows how layoffs changed across different years in different companies from different countries
- Some companies contributed more heavily to total layoffs than others
- Layoffs were concentrated in specific countries, companoies and industries
- Layoffs by year helps to understand in which year more layoffs happened and monthly layoffs helps to identify peak layoffs period in a year.
